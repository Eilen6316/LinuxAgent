# 行为准则

LinuxAgent 遵循 Contributor Covenant 行为准则 2.1 版。

## 我们的承诺

我们承诺让每个人都能在无骚扰的环境中参与项目，不因年龄、体型、可见或不可见
的残障、族裔、性特征、性别认同和表达、经验水平、教育背景、社会经济地位、
国籍、外貌、种族、宗教或性取向而受到区别对待。

## 我们的标准

有助于建立积极环境的行为包括：

- 使用友好、包容的语言。
- 尊重不同观点和经验。
- 提供并接受建设性反馈。
- 聚焦对社区和用户最有利的事情。
- 对其他社区成员保持同理心。

不可接受的行为包括：

- 骚扰、威胁、侮辱或贬低性评论。
- 公开或私下的人身攻击。
- 未经明确许可公开他人隐私信息。
- 挑衅或持续扰乱项目讨论。
- 任何在专业环境中会被合理认为不合适的行为。

## 执行

项目维护者可以删除、编辑或拒绝不符合本准则的评论、commit、代码、wiki 编辑、
issue 和其他贡献。维护者也可以对不当、威胁性、冒犯性或有害行为采取临时或
永久封禁。

## 报告

行为准则问题请联系仓库维护者。安全漏洞请按 [安全政策](SECURITY.md)
处理，不要公开讨论。
