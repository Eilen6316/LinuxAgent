# LinuxAgent v4.0.0

LinuxAgent v4.0.0 是重写后的第一个正式版本。它用基于 LangGraph 的、
可审计、策略驱动的 CLI 替代旧原型，重点解决需要人工批准的 Linux 运维命令执行。

## 亮点

- LangGraph 状态机，包含显式 HITL 确认节点。
- 能力驱动策略引擎，输出 `SAFE`、`CONFIRM`、`BLOCK`、risk score、
  capabilities 和 matched rules。
- 策略评估前先校验结构化 JSON `CommandPlan`。
- 内置 11 个 YAML runbook，作为 planner guidance 提供给规划器。
- 多步骤 planner 输出，每一步都重新经过策略检查。
- SSH 集群执行使用 `RejectPolicy`、known-host 校验和远程 shell 语法防护。
- Hash-chained audit log，并提供 `linuxagent audit verify`。
- LLM 分析前对输出进行 guard 和脱敏。
- 本地 telemetry spans 和 trace IDs。
- 可复现 release constraints 与 wheel data 验证。

## 兼容性

这是相对 v3 及更早版本的破坏性发布。参见
[迁移指南：v3 到 v4.0.0](../migration-v3-to-v4.md)。

## 验证

Release 门禁包括：

- `make lint`
- `make type`
- `make test`
- `make security`
- `make harness`
- `make integration`
- `make optional-anthropic`
- `python -m pip check`
- `make verify-build`

当前文档验证基线：518 个单元测试、安装 Anthropic extra 后可运行兼容验证、
HITL harness 场景、8 个集成冒烟测试，单元测试覆盖率 87.45%。
