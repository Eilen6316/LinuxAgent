# LinuxAgent v4.1.0

LinuxAgent v4.1.0 是一次安全深度版本。它把 v4 的命令安全边界做得更容易被攻击、
被度量、被验证，也更容易被其他 agent 客户端复用。

## 亮点

- 红队策略 harness，覆盖 24 个命令 agent 攻击用例。
- Shell 结构策略分析，覆盖 pipeline、subshell、command substitution、
  redirect 和嵌套 shell 执行。
- 确定性 LOLBin 与 interpreter escape 检测，覆盖 network-to-shell pipeline、
  `find -exec`、`xargs`、`awk system()`、编辑器 escape 和 interpreter inline
  execution。
- 面向 shell 结构解析的 Hypothesis fuzzing。
- Policy benchmark 报告，包含 P50/P95/P99 延迟指标。
- 可选 HTTP audit sink，保持本地先追加审计语义，并把 sink 投递失败写回本地审计链。
- Telemetry exporter 模式：local JSONL、console、OTLP HTTP JSON 和 none。
- Landlock sandbox 设计，覆盖能力探测、降级顺序、兼容性限制和实现切分。
- 只读 stdio MCP server prototype，提供 policy classify 和 audit verify 工具。

## 安全边界

MCP prototype 刻意保持只读。它不暴露命令执行、文件 patch apply、SSH fan-out 或
secrets。未来如果加入执行能力，必须走与 CLI 相同的 CommandPlan、policy、HITL、
sandbox metadata、audit 和 telemetry 路径。

本版本中的 Landlock 工作是设计文档，不改变默认 sandbox runner 行为。

## 验证

Release 门禁包括：

- `make lint`
- `make type`
- `make security`
- `make red-team`
- `make test`
- `make sandbox`
- `make harness`
- `make integration`
- `make verify-build`

当前文档验证基线：639 个单元测试、24 个红队用例、10 个集成冒烟测试，单元测试覆盖率
86.40%。
