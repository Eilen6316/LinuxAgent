# change 记录模板

**文件名约定**：`YYYY-MM-DD-slug.md`（例如 `2026-04-23-adopt-langgraph.md`）

---

## 变更模板

```markdown
# {{变更标题}}

- **日期**：YYYY-MM-DD
- **类型**：[设计变更 / 参数调整 / 偏离计划 / 规则修订]
- **影响范围**：[涉及的 Plan / rule / design 文件]
- **决策者**：{{姓名或角色}}

## 背景

{{为什么要做这个变更？之前是什么状态？}}

## 新决策

{{具体改成什么？引用哪个文档的哪一节？}}

## 影响

- **受影响文档**：
  - `plan/PlanX.md` §Y
  - `rule/...`
- **受影响代码**：尚未实现 / 需回改哪些模块

## 是否向后兼容

{{是 / 否。如果否，迁移步骤。}}
```
